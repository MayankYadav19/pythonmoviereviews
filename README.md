# pythonmoviereviews
"A Python application that fetches movie details using the OMDB API and stores user reviews in a local SQLite database. Provides a simple command-line interface to search for movies and save reviews."
# Installation
1. Make sure python 3 is installed.
2. clone this repository to your local machine.
# Usage
- when prompted, type the movie name.
- enter your review when asked
- the program will save both the movie details and your review in the database
# Features
- Fetches movie title, year, plot, and cast using the OMDB API.
- Stores movie details and user reviews in a local SQLite database.
- Simple command-line interface.
- Allows you to add a review for any movie.
